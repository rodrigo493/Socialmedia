# Tarefa: Criar Roteiro de Reel

## Contexto de Entrada

Antes de gerar o roteiro, leia:
1. `squads/live-social-media/output/research-results.md` — tema, dor do ICP, ângulo de conteúdo
2. `squads/live-social-media/output/tone-of-voice.md` — tom selecionado para esta publicação

## Processo

### Passo 1 — Selecionar Tom
Com base em `tone-of-voice.md`, escolha entre:
- **Provocativo** — abre com afirmação que contradiz crença comum do dono de studio
- **Humor→Seriedade** — começa leve/irônico, vira para dado/impacto real (padrão @rafael.voll)
- **Autoridade Direta** — fala como consultor que já viu o erro acontecer

### Passo 2 — Definir Duração
- 15s → awareness, provocação simples
- 30s → educação rápida com dado + CTA
- 60s → storytelling, comparativo ou demonstração técnica

### Passo 3 — Escrever Roteiro
Use a estrutura obrigatória:

```
HOOK (0–3s): [Fala + legenda queimada + instrução visual]
SETUP (3–15s): [Contexto da dor ou situação]
DELIVERY (15–45s): [Solução, dado, comparativo ou demonstração]
CTA (últimos 5s): [Comente [PALAVRA] para receber X via DM]
```

### Passo 4 — Gerar Caption e Hashtags
Caption separada do script: gancho textual + desenvolvimento + CTA espelhado do vídeo + hashtags.
