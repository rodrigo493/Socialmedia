# Tarefa: Criar Carrossel de Feed

## Contexto de Entrada

Antes de gerar o carrossel, leia:
1. `squads/live-social-media/output/research-results.md` — tema, dor do ICP, ângulo argumentativo
2. `squads/live-social-media/output/tone-of-voice.md` — tom e nível de provocação

## Processo

### Passo 1 — Selecionar Formato
Escolha o formato mais adequado ao tema:
- **Q&A Dialógico** — Pergunta (slide ímpar) + Resposta (slide par). Melhor para objeções e dúvidas comuns.
- **Comparativo** — Lado A vs Lado B ao longo dos slides. Melhor para posicionamento competitivo.
- **Passo a Passo Argumentativo** — Sequência de raciocínio com cada slide aprofundando o anterior. Melhor para educação.

### Passo 2 — Estruturar os Slides
Quantidade: 6–10 slides. Estrutura obrigatória:
- **Slide 1 (Capa):** Headline de tensão + instrução visual premium
- **Slides 2–N-1:** Conteúdo com progressão argumentativa
- **Slide N (Fechamento):** CTA com "Comente [PALAVRA]"

### Passo 3 — Escrever Cada Slide
Para cada slide, preencha todos os campos obrigatórios (Headline, Texto de Suporte, Palavras de Destaque, Fundo, Foto/Visual).

### Passo 4 — Caption e Hashtags
Caption com gancho textual forte + síntese do argumento + CTA espelhado do último slide + hashtags.
